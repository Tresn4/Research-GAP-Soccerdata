Proyek replikasi ini menggunakan dataset Premier League yang bersumber dari Kaggle sebagai basis data utama. Mengingat penelitian referensi (Nadifa) tidak mencantumkan tautan atau detail spesifik mengenai dataset yang digunakan, saya menggunakan dataset yang telah saya verifikasi isinya identik secara statistik dengan tabel data yang ditampilkan dalam paper tersebut, sehingga dapat dipastikan ini adalah dataset yang valid dan seharusnya digunakan oleh penulis asli untuk menghasilkan output yang presisi.
